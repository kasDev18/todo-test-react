## Shiftleft Laravel Starter
- Current Version: Laravel 8
- Basic Modules:
  - Users Management
  - Role Management

## Installation
1. ```composer install```
2. ```npm install```
3. Import /database/starter_db.sql as initial database
4. ```php artisan migrate```
5. ```php artisan db:seed```
